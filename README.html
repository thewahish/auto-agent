AutoAgent: Fully-Automated LLM Agent Framework

Domain: Research-Development | Status: Active

A Fully-Automated and highly Self-Developing framework that enables users to create and deploy LLM agents through Natural Language Alone.

Key Features

  • Natural Language-Driven: Build agents through dialogue, no coding required
  • Zero-Code Framework: Anyone can create and customize agents
  • Self-Managing Workflows: Dynamic workflow generation and optimization
  • Intelligent Orchestration: Controlled code generation for tools and agents
  • Self-Play Customization: Iterative self-improvement capabilities

Modes

  • User Mode: Deep Research Agents - Ready-to-use multi-agent system
  • Agent Editor: Create agents without workflow using natural language
  • Workflow Editor: Create agent workflows with natural language

Quick Start

# Installation
git clone https://github.com/HKUDS/AutoAgent.git
cd AutoAgent
pip install -e .

# Start AutoAgent
auto main
